# 隐私相册（伪装计算器）需求文档

## 1. 项目概述

一款伪装成计算器的隐私相册应用。表面上是普通计算器，输入正确密码后可进入隐藏目录，查看和管理私密照片/视频。

- **目标设备**：一加 Ace 2 Pro，ColorOS 16
- **技术栈**：原生 Android（Kotlin）
- **使用范围**：仅个人使用，单设备

## 2. 伪装要求

| 项目 | 说明 |
|------|------|
| App 名称 | Calculator |
| App 图标 | 计算器样式（待提供素材或使用 Vector Drawable） |
| 多任务栏 | 正常显示，不做隐藏处理 |

## 3. 计算器功能

- 支持基本四则运算（加、减、乘、除）
- 优先使用系统自带的计算方法（如 `BigDecimal`）
- 界面模仿系统计算器风格，外人看不出破绽

## 4. 密码机制

| 项目 | 说明 |
|------|------|
| 密码格式 | 纯数字（不含运算符） |
| 确认方式 | 按 `=` 键确认，如输入 `1234=` |
| 输入过程 | 正常显示计算过程，按 `=` 后判断是否为密码 |
| 密码错误 | 无特殊处理，不防暴力破解 |
| 密码管理 | 进入隐藏目录后可查看（明文）和修改密码 |

## 5. 隐藏目录功能

进入隐藏目录后，需支持以下操作：

### 5.1 导入
- 从手机相册选择文件导入
- 支持批量选择

### 5.2 导出
- 导出到手机相册
- 支持批量选择

### 5.3 删除
- 支持批量删除

### 5.4 浏览
- 列表页显示缩略图
- 支持大图/视频查看器（缩放、滑动切换）

## 6. 支持的媒体格式

| 类型 | 格式 |
|------|------|
| 图片 | JPG、PNG、HEIC、GIF |
| 视频 | MP4、AVI |

## 7. 存储方案

| 项目 | 说明 |
|------|------|
| 存储位置 | App 私有目录（`/data/data/包名/` 或 `Android/data/包名/`） |
| 加密 | 不需要，仅靠密码作为入口 |
| 隐蔽性 | ✅ 手机图库看不到 |
|  | ✅ 文件管理器普通用户找不到 |
|  | ⚠️ Root 用户可以找到（可接受） |

## 8. 不需要的功能

- ❌ 引导页
- ❌ 存储容量提示
- ❌ 密码找回机制
- ❌ 防暴力破解
